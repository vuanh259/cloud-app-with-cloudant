# Car Dealership Portal

## Repository Name
- `cloud-app-with-cloudant`

## Project Name
- **National Car Dealership and Reviews Management System**

### Description
A full-stack web application built using Django, React, Node.js, and MongoDB. The platform allows users to browse car dealerships across various states, view customer reviews, perform sentiment analysis on reviews, and submit their own dealership feedback.

### CI/CD Deployment
- Managed via GitHub Actions
- Deployed on IBM Cloud Code Engine
